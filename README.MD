
# ![](./assets/logo_en.png)

**Современная онлайн-игра Судоку с поддержкой мультиплеера, таймера, ошибок, статистики пользователей и локализации.  
Разработано на Phaser + Node.js + Socket.IO.**

## 🚀 Особенности проекта

### 🎮 Игровой процесс
- Классическое Sudoku 9×9
- Три уровня сложности: лёгкий, средний, сложный
- Система штрафов и **лимит ошибок**
- Подсветка выбранных чисел и клеток
- Таймер
- Оптимизированный UI под браузер

### 🧑‍🤝‍🧑 Мультиплеер (Socket.IO)
- Поиск второго игрока при входе
- Два активных игрока на одном поле
- Синхронизация всех действий в реальном времени
- Проверка заполнения поля у всех игроков
- Автоматическое завершение раунда при победе любого участника

### 🌍 Локализация
- Поддержка русского и английского языка
- Автоматическая загрузка языка через конфиг
- Динамический текст интерфейса

### 📊 Статистика пользователей
- Никнеймы через куки
- Хранение данных в `users.json`:
    - победы
    - поражения
    - сыгранные игры
- Автосоздание пользователя при первом входе

# 🏗 Установка и запуск

## Docker
```
git clone https://github.com/username/sudoku-multiplayer.git
cd sudoku-multiplayer

docker-compose up --build
```

## Без docker

```
npm install
node server.js
```
